Fake News Detector

The Fake News Detector is a Machine Learning project developed using Python to identify whether a news article is REAL or FAKE. The system uses Natural Language Processing (NLP) techniques to analyze textual news content and classify it using a Logistic Regression algorithm. TF-IDF Vectorization is applied to convert text data into numerical format so that the machine learning model can process and learn patterns from the dataset.

This project is implemented using libraries such as pandas and scikit-learn. The dataset contains news articles along with labels indicating whether the news is fake or real. After training the model, users can enter any news headline or article, and the system predicts its authenticity. The project demonstrates the application of machine learning in detecting misinformation and preventing the spread of fake news on digital platforms.
